**Prompt 1: Stakeholder Performance Domain**

"Based on the content of the uploaded project management texts, generate 100 unique multiple-choice questions (MCQs) covering the core principles, key concepts, processes (such as stakeholder identification, analysis, engagement, and communication management), and common practices related to the **Stakeholder Performance Domain**.

The output must be a single JSON object.
The root of the JSON object should have two keys: `\"topic\"` and `\"questions\"`.
The value for `\"topic\"` should be \"Stakeholder Performance Domain Quiz\".
The value for `\"questions\"` should be an array of 100 unique MCQ objects.

Each MCQ object in the `\"questions\"` array must have the following structure:
- `\"questionText\"`: A string containing the question.
- `\"choices\"`: An array of exactly four strings: one correct answer and three plausible distractors.
- `\"correctAnswer\"`: An integer representing the 0-based index of the correct answer within the `\"choices\"` array.
- `\"explanation\"`: A string providing a detailed explanation for why the correct answer is right.
- `\"time\"`: An integer representing the allocated time for the question in seconds (e.g., 30).

Ensure all 100 questions are unique, suitable for PMP exam preparation, and reflect a comprehensive understanding of the Stakeholder Performance Domain as presented in the uploaded sources."
